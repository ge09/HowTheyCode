ABOUT THIS

'How They Code' is a software development company search app, where you can see how well (or bad) does a company work depending on some surveys employees or ex-employees fill (using the Joel's test).

If your company is not registered, you can create it and fill the survey yourself. If it is already registered, just go and answer the questions.

This site is based on faith in people, so it's up to you if you trust it or not. It's not a place to troll, it's just a place to get some information.

You can have a look or fork this proyect on Github

    These are the 12 questions

  • Do you use source control?
  • Can you make a build in one step?
  • Do you make daily builds?
  • Do you have a bug database?
  • Do you fix bugs before writing new code?
  • Do you have an up-to-date schedule?
  • Do you have a spec?
  • Do programmers have quiet working conditions?
  • Do you use the best tools money can buy?
  • Do you have testers?
  • Do new candidates write code during their interview?
  • Do you do hallway usability testing?
